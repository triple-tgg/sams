# 01. Project Overview

## 1. บทนำ (Introduction)
เอกสารฉบับนี้จัดทำขึ้นเพื่อแสดงภาพรวมของ **SAMS Module QA (Quality Assurance)** ซึ่งเป็นส่วนหนึ่งของระบบ **SAM Airline Maintenance System (SAMS)** โดยมุ่งเน้นการพัฒนาระบบบริหารจัดการข้อมูลและการปฏิบัติตามข้อกำหนด (Compliance) สำหรับบุคลากรฝ่ายประกันคุณภาพ (QA) และช่างซ่อมบำรุงอากาศยาน (Certifying Staff) เพื่อให้สอดคล้องกับมาตรฐานความปลอดภัยด้านการบินทั้งในและต่างประเทศ (เช่น CAAT, EASA Part-145)

## 2. วัตถุประสงค์ (Objectives)
- **ศูนย์กลางข้อมูล (Centralization):** เพื่อรวมศูนย์ข้อมูลบุคลากร, ประวัติการฝึกอบรม (Training Records), และการได้รับอนุมัติให้ปฏิบัติงาน (Authorization) ไว้ในที่เดียว
- **การติดตามผลแบบเรียลไทม์ (Real-time Monitoring):** เพื่อให้สามารถตรวจสอบสถานะวันหมดอายุของ Training และ Authorization ได้ล่วงหน้าผ่าน Dashboard และ Matrix อย่างชัดเจน
- **ลดข้อผิดพลาด (Error Reduction):** ลดความผิดพลาดจากกระบวนการ Manual เช่น การตรวจสอบวันหมดอายุจากเอกสาร Excel/PDF หลายไฟล์
- **การตรวจสอบย้อนหลัง (Audit Trail):** สร้างความโปร่งใสในการอนุมัติสิทธิ์ (Authorization) และสามารถตรวจสอบประวัติการต่ออายุได้

## 3. ผู้ที่ได้รับประโยชน์ (Stakeholders / Beneficiaries)
1. **QA Manager & Supervisor:** สามารถติดตามสถานะ (Monitoring) ภาพรวมของบุคลากรทั้งหมด อนุมัติสิทธิ์ และจัดตารางการฝึกอบรมได้อย่างมีประสิทธิภาพ
2. **Trainer / Training Coordinator:** จัดการหลักสูตร (Course Management) และบันทึกประวัติการเข้าเรียน (Training Records) ได้สะดวกรวดเร็ว
3. **Certifying Staff / Engineer:** มั่นใจได้ว่าข้อมูลการรับรองและวันหมดอายุการฝึกอบรมของตนเองมีความถูกต้องและเป็นปัจจุบันเสมอ
4. **Auditor (Internal / External):** สามารถตรวจสอบข้อมูลและหลักฐานการผ่านการอบรม/การอนุญาต (Compliance) ได้ง่ายและเป็นระบบ

## 4. ขอบเขตของระบบ (Scope of the System)

### สิ่งที่ระบบนี้ "ทำ" (In-Scope)
1. **QA Staff Management:** จัดการฐานข้อมูลรายชื่อพนักงานในส่วน QA
2. **Training Record & Monitoring:** 
   - จัดการข้อมูลหลักสูตร (Course Management)
   - จัดการตารางเรียน (Training Scheduler)
   - ติดตามสถานะการผ่านการฝึกอบรม (QA Training Matrix)
3. **Authorization Monitoring:**
   - ตรวจสอบสถานะและวันหมดอายุของการได้รับสิทธิ์จากสายการบินลูกค้า (Customer Authorization)
   - ตรวจสอบสถานะการได้รับสิทธิ์จากหน่วยงานกำกับดูแล (Authority Authorization)
   - ระบบการจัดการ Matrix อนุมัติสิทธิ์ของพนักงาน

### สิ่งที่ระบบนี้ "ไม่ทำ" (Out-of-Scope)
1. **Payroll / การจ่ายเงินเดือน:** ระบบจะไม่คำนวณหรือจัดการเรื่องค่าตอบแทนของพนักงาน
2. **HR Recruitment:** ระบบจะไม่ครอบคลุมกระบวนการสรรหาพนักงานใหม่
3. **Flight Operations / Scheduling:** ไม่เกี่ยวกับการจัดตารางการบิน หรือการทำงานของลูกเรือ (Cabin Crew)
4. **Inventory Management:** ระบบนี้ไม่จัดการเรื่องอะไหล่ซ่อมบำรุงอากาศยาน (Parts / Materials)

## 5. สรุปคุณค่าของระบบ (Value Proposition)
ระบบ SAMS Module QA จะเปลี่ยนผ่านรูปแบบการทำงานจากการใช้เอกสาร (Paper-based) หรือ Spreadsheet กระจัดกระจาย มาสู่ความเป็น **Digital Platform** ที่มีความน่าเชื่อถือสูง ช่วยลดความเสี่ยงในการดำเนินงานโดยไม่ได้รับอนุญาต (Non-compliance risks) และยกระดับมาตรฐานความปลอดภัยในการซ่อมบำรุงอากาศยานขององค์กร
